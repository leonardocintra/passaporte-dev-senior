## Exemplo de "Procedural com Classes" vs "Orientado à Objetos"

- Procedural com Classes

```python
apples = [20, 4, 5, 10, 15]

class Sorted:
    def __init__(self, items):
        self.data = items

    def sort(self):
        self.data.sort()

    def get(self, index):
        return self.data[index]


def consumer1(cesta):
    """Pega a maçã mais pessada."""
    s = Sorted(cesta)
    s.sort()
    return s.get(-1)

print(consumer1(apples))
```

- Orientado à Objetos

```python
apples = [20, 4, 5, 10, 15]


class Balança:
    def __init__(self, items):
        self.data = items

    def mais_pesado(self):
        return max(self.data)


def consumer2(cesta):
    """Pega a maçã mais pessada."""
    s = Balança(cesta)
    return s.mais_pesado()


print(consumer2(apples))
```
